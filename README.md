# Ejercicio Nº 4

# Inicio: Diff21

## Ingreso de datos

- numero
- resultado

## Proceso

- Si el número ingresado es menor de 21, entonces 
    - Obtener la resta de 21-numero;
    - mostrar el resultado
- Caso contrario
    - Obtener el doble de la resta del numero menos 21, 2*(numero-21)
    - mostrar el resultado

*Fin*
